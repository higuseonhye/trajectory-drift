# Recovery

Recovery mechanisms for trajectory restoration — reducing noise, rebuilding coherence, increasing leverage.

Recovery is **operational**, not motivational. Each mechanism connects to measurable behavior change.

---

## Recovery mechanisms

### Reduce noise

- Cut feed and trend inputs
- Batch reactive communication
- Protect deep work blocks
- Eliminate low-leverage commitments

**Addresses:** Noise Drift, reactive behavior signals.

---

### Restore ownership

- Shift time from labor to asset-building
- Ship owned work products
- Build reusable systems
- Convert skills to assets

**Addresses:** Labor Drift, Apprentice Drift, Prestige Drift.

---

### Rebuild coherence

- Revisit long-term goals
- Align time allocation with stated values
- Close or kill fragmented projects
- Restore cross-domain balance

**Addresses:** Spiritual Drift, fragmented direction.

See [trajectory-native/framework/coherence/](https://github.com/higuseonhye/return/tree/main/framework/coherence).

---

### Increase leverage

- Build systems that persist
- Create owned distribution
- Automate repeatable work
- Delegate with clear propagation

**Addresses:** Labor Drift, low-leverage activity.

---

### Compound systems

- Document decisions and rationale
- Build on prior work instead of resetting
- Create institutional memory
- Review outcomes and update judgment

**Addresses:** Decision entropy, organizational memory loss.

---

### Trajectory reflection

- Structured review of trajectory events
- Decision journal outcome review
- Drift pattern recognition
- Calibration logging

**Addresses:** All drift categories — closes the reality loop.

---

### Long-term projects

- Commit to multi-year compounding work
- Reduce project switching
- Build assets that accumulate
- Accept short-term visibility cost for long-term leverage

**Addresses:** Fragmentation, prestige drift, apprentice drift.

---

## Product connection

Intervention layer in trajectory-native surfaces recovery actions when drift is detected — not passive insight.

Calibration journal in trajectory-drift records what stabilized after recovery.
