# Drift signals

Observable indicators that drift is occurring — behavior-linked, measurable, workflow-connected.

---

## Activity signals

| Signal | Indicates |
|--------|-----------|
| Fragmented projects | No leverage accumulation; context switching |
| Reactive behavior | Noise drift; urgency-driven decisions |
| Shallow networking | Prestige drift; transactional relationships |
| Endless consumption | Apprentice drift; learning without application |
| Low ownership | Labor drift; optimized worker pattern |
| No leverage accumulation | Activity without compounding |
| No reusable systems | Resetting each project |
| Constant context switching | Noise drift; fragmented direction |

---

## Ratio signals

| Ratio | Drift when skewed toward… |
|-------|---------------------------|
| Consumption vs creation | Consumption |
| Networking vs ownership-building | Networking |
| Applications vs building | Applications |
| Learning vs execution | Learning |
| Meetings vs closed loops | Meetings |
| Tool adoption vs system building | Tool adoption |

---

## Temporal signals

- Goals stated but not revisited
- Decisions without outcome review
- Patterns repeating across quarters
- High activity, flat asset accumulation
- Increasing urgency, decreasing coherence

---

## Organizational signals

- Meeting entropy rising
- Alignment decay between teams
- Incidents repeating without pattern resolution
- Institutional memory not referenced in decisions
- Execution theater — activity without propagation

See [`../organizational-drift/`](../organizational-drift/).
