# Environment design (org scale)

Dead environments create drift at team and civilization scale — not only cognitive failure.

Personal-layer canonical docs: [trajectory-native/framework/environment-design/](https://github.com/higuseonhye/return/tree/main/framework/environment-design)

---

## Principles

- **Execution theater in sterile spaces** is environmental drift
- **Always-on digital numbness** suppresses signal and judgment
- **Atmosphere affects coordination** — social energy, warmth, honesty
- Recovery includes designing **anti-drift atmospheres** for how teams work

---

## Product implications (trajectory-drift)

- Flag environmental drift patterns in behavioral signals
- Pair with native bridge for individual environment context on events
- Institutional memory: document not only *what* failed but *where* culture went dead
