# Physical layer (org scale)

Software navigates trajectory; **physical and cultural environments** carry it at team scale.

Canonical personal layer: [trajectory-native/framework/physical-layer/](https://github.com/higuseonhye/trajectory-native/tree/main/framework/physical-layer)

---

## Org-scale domains

| Domain | Anti-drift role |
|--------|-----------------|
| **Workspace atmosphere** | Reduce fluorescent execution theater |
| **Sync vs async rhythm** | Prevent scroll-loop coordination numbness |
| **Hospitality / offsites** | Restore awe, scale, honest contact |
| **Ritual** | Embodied steering practices for teams |
| **Spatial design** | Curated spaces that restore aliveness |
